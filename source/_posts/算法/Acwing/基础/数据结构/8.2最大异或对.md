[143. 最大异或对 - AcWing题库](https://www.acwing.com/problem/content/145/)


### 题目描述
在给定的 $N$ 个整数 $A_1，A_2……A_N$ 中选出两个进行 $xor$（异或）运算，得到的结果最大是多少？

#### 输入格式

第一行输入一个整数 $N$。

第二行输入 $N$ 个整数 $A_1$～$A_N$。

#### 输出格式

输出一个整数表示答案。

#### 数据范围

$1 \le N \le 10^5$,  
$0 \le A_i < 2^{31}$

#### 输入样例：

```
3
1 2 3
```

#### 输出样例：

```
3
```

---
### 算法

```cpp
#include<iostream>
#include<algorithm>
using namespace std;
const int N = 1e5 + 10;
int son[N * 30][2], idx;
int n, a[N];

void insert(int x)
{
    int p = 0;
    for (int i = 30; ~i; i--)
    {
        int u = x >> i & 1;
        if (!son[p][u]) son[p][u] = ++idx;
        p = son[p][u];
    }
}

int query(int x)
{
    int p = 0, ans = 0;
    for (int i = 30; ~i; i--)
    {
        int u = x >> i & 1;
        if (son[p][!u])
        {
            ans = (ans << 1) + !u;
            p = son[p][!u];
        }
        else
        {
            ans = (ans << 1) + u;
            p = son[p][u];
        }
    }
    return ans;
}

int main()
{
    cin >> n;
    for (int i = 0; i < n; i++)
    {
        cin >> a[i];
        insert(a[i]);
    }
    int res = -1;
    for (int i = 0; i < n; i++)
    {
        res = max(res, a[i] ^ query(a[i]));
    }
    cout << res << endl;
}
```
