[895. 最长上升子序列 - AcWing题库](https://www.acwing.com/problem/content/897/)


### 题目描述
给定一个长度为 $N$ 的数列，求数值严格单调递增的子序列的长度最长是多少。

#### 输入格式

第一行包含整数 $N$。

第二行包含 $N$ 个整数，表示完整序列。

#### 输出格式

输出一个整数，表示最大长度。

#### 数据范围

$1 \le N \le 1000$，  
$-10^9 \le 数列中的数 \le 10^9$

#### 输入样例：

```
7
3 1 2 1 8 5 6
```

#### 输出样例：

```
4
```

### 算法

![](最长上升子序列/Pasted%20image%2020240513001939.png)

```cpp
#include<iostream>
const int N = 1010;
int a[N];
int f[N];
int n;

int main()
{
    std::cin >> n;
    for (int i = 1; i <= n; i++) std::cin >> a[i];
    int res = 1;
    for (int i = 1; i <= n; i++)
    {
        f[i] = 1;
        for (int j = 1; j < i; j++)
        {
            if (a[i] > a[j])
            {
                f[i] = std::max(f[i], f[j] + 1);
            }
        }
        res = std::max(res, f[i]);
    }
    std::cout << res << std::endl;
}
```
