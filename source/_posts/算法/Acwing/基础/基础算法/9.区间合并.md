[803. 区间合并 - AcWing题库](https://www.acwing.com/problem/content/805/)


### 题目描述
给定 $n$ 个区间 $[l_i, r_i]$，要求合并所有有交集的区间。

注意如果在端点处相交，也算有交集。

输出合并完成后的区间个数。

例如：$[1,3]$ 和 $[2,6]$ 可以合并为一个区间 $[1,6]$。

#### 输入格式

第一行包含整数 $n$。

接下来 $n$ 行，每行包含两个整数 $l$ 和 $r$。

#### 输出格式

共一行，包含一个整数，表示合并区间完成后的区间个数。

#### 数据范围

$1 \le n \le 100000$,  
$-10^9 \le l_i \le r_i \le 10^9$

#### 输入样例：

```
5
1 2
2 4
5 6
7 8
7 9
```

#### 输出样例：

```
3
```

---
### 算法

```cpp
#include<iostream>
#include<algorithm>
#include<vector>
using namespace std;
const int N = 100010;
typedef pair<int, int> PII;

vector<PII> segs;

void merge(vector<PII>& segs)
{
    vector<PII>res;     //存合并后的
    //对区间左端点排序
    sort(segs.begin(), segs.end());     //默认左排序
    int st = -1e9-10, ed = -1e9-10;
    for (auto seg : segs)
    {
        if (ed < seg.first)
        {
            //加入到答案
            if (st != -1e9-10)res.push_back({ st, ed });
            st = seg.first, ed = seg.second;
        }
        else
        {
            ed = seg.second;
        }
    }
    //把最后一个加进去
    if (st != -1e9-10)res.push_back({ st, ed });
    segs = res;
}



int main()
{
    int n;
    cin >> n;
    for (int i = 0; i < n; i++)
    {
        int l, r;
        cin >> l >> r;
        segs.push_back({ l, r });
    }
    merge(segs);
    cout << segs.size() << endl;
    return 0;
}
```
