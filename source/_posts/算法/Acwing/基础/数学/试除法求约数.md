[869. 试除法求约数 - AcWing题库](https://www.acwing.com/problem/content/871/)

### 题目描述
给定 $n$ 个正整数 $a_i$，对于每个整数 $a_i$，请你按照从小到大的顺序输出它的所有约数。

#### 输入格式

第一行包含整数 $n$。

接下来 $n$ 行，每行包含一个整数 $a_i$。

#### 输出格式

输出共 $n$ 行，其中第 $i$ 行输出第 $i$ 个整数 $a_i$ 的所有约数。

#### 数据范围

$1 \le n \le 100$,  
$1 \le a_i \le 2 \times 10^9$

#### 输入样例：

```
2
6
8
```

#### 输出样例：

```
1 2 3 6 
1 2 4 8 
```

---
### 算法

```cpp
#include<iostream>
#include<vector>
#include<algorithm>
using namespace std;

vector<int>res;
void getdivisor(int x)
{
    for (int i = 1; i <= x / i; i++)
    {
        if (x % i == 0)
        {
            res.push_back(i);
            if (x / i != i) res.push_back(x / i);
        }
    }
    sort(res.begin(), res.end());
}



int main()
{
    int n;
    cin >> n;
    while (n--)
    {
        int x;
        cin >> x;
        getdivisor(x);
        for (auto item : res) cout << item << " ";
        cout << endl;
        res.clear();
    }
    return 0;
}
```
