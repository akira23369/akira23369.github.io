# 缩放
```csharp
// 相对于世界坐标系     // 不能改
print(this.transform.lossyScale);

// 相对于父对象        // 可以改
print(this.transform.localScale);

// unity 没有提供缩放相关的API, 只能自己手动来搞    eg: 
this.transform.localScale += Vector3.one * Time.deltaTime;
```

# 看向
```csharp
// 将自己的面朝向源点，相对于世界坐标系
this.transform.LookAt(Vector3.zero);
```
